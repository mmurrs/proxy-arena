What this unlocks

Signed leaderboards

Scores strangers can check. A competition where anyone can submit an entry because a fake score is cryptographically impossible — no human review required.

Automatic payouts

A prize pool that pays the top-3 automatically on signed scores — no dispute process. The same pattern as ecdsa.fail (Eigen Labs' own open challenge) but with receipts for every result.

Results that travel

A lab cites an external benchmark number in a model card because the run has a receipt. One operator's score means something to another lab without re-running, because it's checkable.

Receipts prove an eval ran faithfully — same replay number (seed) + same decisions = identical match, so anyone can re-run and check. They do not prove the eval is well-designed — game-design quality stays a human problem.
